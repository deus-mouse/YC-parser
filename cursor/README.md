# Парсер свободного времени мастеров YClients

Парсер сайта онлайн-записи. Собирает информацию о свободном времени каждого мастера на заданную глубину (дней).

## Требования

- Python 3.10+
- Chrome браузер
- ChromeDriver (совместимый с версией Chrome)

## Установка

Рекомендуется использовать виртуальное окружение:

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
```

## Запуск

```bash
# С параметрами по умолчанию (URL и 30 дней)
python main.py

# С указанием URL и глубины сканирования
python main.py --url "https://n625088.yclients.com/company/266762/personal/select-master?o=" --depth 30
```

## Входные данные

- **url** — ссылка на страницу выбора мастера
- **depth** — глубина сканирования в днях (по умолчанию 30)

## Выходные данные

Список мастеров с количеством свободных минут в течение указанного периода. Свободные минуты считаются как: количество доступных слотов × длительность самой короткой услуги мастера.

## Алгоритм

1. Загрузка страницы с мастерами
2. Для каждого мастера:
   - Переход на страницу услуг
   - Выбор самой короткой по времени услуги
   - Переход в календарь
   - Обход всех рабочих дней на глубину сканирования
   - Подсчёт свободных слотов в каждом дне
   - Суммирование свободных минут
3. Вывод отсортированного списка мастеров
